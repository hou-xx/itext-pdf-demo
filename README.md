# 借助 itext 实现 java 代码填充 PDF 表单

## 引用 jar 包
- itextpdf
> maven 引用
```aidl
<dependency>
    <groupId>com.itextpdf</groupId>
    <artifactId>itextpdf</artifactId>
    <version>5.4.3</version>
</dependency>
```
- itext-asian （中文时需要）
> maven 引用
```aidl
<dependency>
    <groupId>com.itextpdf</groupId>
    <artifactId>itext-asian</artifactId>
    <version>5.2.0</version>
</dependency>
```
